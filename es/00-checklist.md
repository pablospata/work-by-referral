# Checklist: Trabajar por Referidos

*Marcá cada ítem a medida que lo completes. Imprimí esta lista y tenela a mano.*

---

## FASE 1: Mentalidad y Fundamentos — Semana 1

- [ ] Hice mi diagnóstico honesto por escrito (5 preguntas)
- [ ] Escribí a mano mi visión del "ambos/y" (día ideal)
- [ ] Definí mis metas y las anoté en una tarjeta de bolsillo:
  - [ ] Meta financiera (ingreso neto anual)
  - [ ] Meta de producción (transacciones)
  - [ ] Meta familiar
  - [ ] Meta personal
  - [ ] Meta de contribución
- [ ] La tarjeta está donde la veo todos los días

---

## FASE 2: Construir la Base de Datos — Semanas 2-3

- [ ] Me senté 1 hora y escribí todos los nombres que conozco
- [ ] Revisé cada categoría de disparadores:
  - [ ] Familia
  - [ ] Amigos
  - [ ] Clientes pasados
  - [ ] Clientes actuales
  - [ ] Vecinos
  - [ ] Servicios que uso
  - [ ] Grupos y asociaciones
  - [ ] Trabajo anterior
  - [ ] Redes sociales
- [ ] Cargué los nombres en un CRM o planilla
- [ ] Llegué a 50 nombres
- [ ] Llegué a 75 nombres
- [ ] Llegué a 100 nombres
- [ ] Todos los contactos tienen nombre, teléfono y dirección
- [ ] Clasifiqué cada nombre (A+, A, B, C o D)
- [ ] Eliminé los "D" de la base de datos

---

## FASE 3: Activar la Base de Datos — Semanas 3-5

- [ ] Elegí mi carta (Introducción o Confesión)
- [ ] Redacté la carta
- [ ] Envié la carta a toda mi base de datos
- [ ] Memoricé el Diálogo de los Tres Grandes
- [ ] Memoricé el Diálogo de la Campaña del Alcalde
- [ ] Memoricé el Diálogo del Agente Nuevo (si aplica)
- [ ] Hice mi primera ronda de llamadas de seguimiento:
  - [ ] Semana 1: 25 llamadas completadas
  - [ ] Semana 2: 25 llamadas completadas
  - [ ] Semana 3: 25 llamadas completadas
  - [ ] Semana 4: 25 llamadas completadas
- [ ] Usé la Campaña del Alcalde al menos 3 veces esta semana
- [ ] Agregué al menos 1 nombre nuevo con la Campaña del Alcalde

## FASE 4: Las Tres C — A partir de la Semana 6

### Contacto

- [ ] Elegí mi primer Elemento de Valor para este mes
- [ ] Envié el Elemento de Valor a toda la base de datos
- [ ] Hice las llamadas de seguimiento post-Elemento de Valor
- [ ] Usé "Ah, por cierto..." en cada llamada
- [ ] Preparé el Elemento de Valor del mes siguiente

### Cuidado — Notas personales

- [ ] Tengo tarjetas y sobres en mi escritorio
- [ ] Establecí el hábito: 3 notas antes del email cada mañana
- [ ] Semana 1: escribí 15 notas
- [ ] Semana 2: escribí 15 notas
- [ ] Semana 3: escribí 15 notas
- [ ] Semana 4: escribí 15 notas
- [ ] Primer mes: 60 notas enviadas (total acumulado)

### Cuidado — Pop-Bys

- [ ] Preparé obsequios para Pop-Bys (primer lote)
- [ ] Tracé una ruta de Pop-Bys compatible con mi recorrido diario
- [ ] Semana 1: hice 7-10 Pop-Bys
- [ ] Semana 2: hice 7-10 Pop-Bys
- [ ] Semana 3: hice 7-10 Pop-Bys
- [ ] Semana 4: hice 7-10 Pop-Bys
- [ ] Cerré cada Pop-By con "Ah, por cierto..."

### Comunidad

- [ ] Elegí mi primer evento de comunidad para organizar:
  - [ ] Fiesta de bienvenida al hogar
  - [ ] Fiesta de apreciación al cliente
  - [ ] Mixer de negocios
  - [ ] Almuerzo de negocios
- [ ] Organicé mi primer evento
- [ ] Hice seguimiento post-evento (llamadas a asistentes)
- [ ] Programé al menos 2 almuerzos de negocios esta semana
- [ ] Completé 8 almuerzos de negocios este mes

## FASE 6: Ganá el Día — Permanente

- [ ] Imprimí y pegué mi Estándar Personal de Negocio

- [ ] Programé mis 2 descansos de 30 minutos

- [ ] Encontré un Socio de Responsabilidad
- [ ] Acordamos día y hora fija semanal para reportar
