# Checklist: Referidos para el Asesor Financiero

*Marcá cada ítem a medida que lo completes.*

---

## FASE 1: Mentalidad y Fundamentos — Semana 1

- [ ] Hice mi diagnóstico honesto por escrito (5 preguntas)
- [ ] Escribí a mano mi visión del "ambos/y" (día ideal)
- [ ] Definí mis metas y las anoté en una tarjeta de bolsillo:
  - [ ] Meta de AUM
  - [ ] Meta de clientes nuevos
  - [ ] Meta de ingreso por comisiones
  - [ ] Meta familiar
  - [ ] Meta personal
- [ ] La tarjeta está donde la veo todos los días

---

## FASE 2: Construir la Base de Datos — Semanas 2-3

- [ ] Me senté 1 hora y escribí todos los nombres que conozco
- [ ] Revisé cada categoría de disparadores:
  - [ ] Familia
  - [ ] Amigos
  - [ ] Clientes actuales
  - [ ] Ex colegas
  - [ ] Profesionales (contador, abogado, médico, etc.)
  - [ ] Vecinos y conocidos
  - [ ] Grupos y actividades
  - [ ] Empresarios / dueños de negocios
  - [ ] Redes sociales (WhatsApp, LinkedIn, Instagram)
- [ ] Cargué los nombres en CRM o planilla
- [ ] Llegué a 50 nombres
- [ ] Llegué a 75 nombres
- [ ] Llegué a 100 nombres
- [ ] Todos tienen nombre y al menos un contacto (teléfono o email)
- [ ] Clasifiqué cada nombre (A+, A, B, C o D)
- [ ] Eliminé los "D"

---

## FASE 3: Activar la Base de Datos — Semanas 3-5

- [ ] Elegí mi mensaje (Presentación o Reconexión)
- [ ] Redacté y personalicé el mensaje
- [ ] Envié el mensaje a toda mi base de datos
- [ ] Memoricé la estructura de los "Tres Grandes"
- [ ] Memoricé el "Diálogo del Networker"
- [ ] Hice mi primera ronda de llamadas/contactos:
  - [ ] Semana 1: 25 contactos
  - [ ] Semana 2: 25 contactos
  - [ ] Semana 3: 25 contactos
  - [ ] Semana 4: 25 contactos
- [ ] Usé el Diálogo del Networker al menos 3 veces esta semana
- [ ] Agregué al menos 1 nombre nuevo con el Diálogo

---

## FASE 4: Las Tres C — A partir de Semana 6

### Contacto

- [ ] Creé mi primer Elemento de Valor (informe, tip, guía)
- [ ] Lo envié a toda la base de datos
- [ ] Hice las llamadas/mensajes de seguimiento
- [ ] Usé "Ah, por cierto..." en cada contacto
- [ ] Preparé el Elemento de Valor del mes siguiente

### Cuidado — Mensajes personales

- [ ] Establecí el hábito: 3 mensajes antes de mirar cotizaciones
- [ ] Semana 1: 15 mensajes personales enviados
- [ ] Semana 2: 15 mensajes personales enviados
- [ ] Semana 3: 15 mensajes personales enviados
- [ ] Semana 4: 15 mensajes personales enviados
- [ ] Primer mes: 60 mensajes personales (total acumulado)

### Cuidado — Cafés uno a uno

- [ ] Agendé mis primeros 2 cafés de la semana
- [ ] Semana 1: hice 2 cafés
- [ ] Semana 2: hice 2 cafés
- [ ] Semana 3: hice 2 cafés
- [ ] Semana 4: hice 2 cafés
- [ ] Cerré cada café con "Ah, por cierto..."

### Comunidad

- [ ] Elegí mi primer evento de comunidad:
  - [ ] Desayuno/almuerzo de mercado
  - [ ] Cena para clientes A+
  - [ ] After office / asado de networking
  - [ ] Almuerzo con aliado profesional
- [ ] Organicé el evento
- [ ] Hice seguimiento individual con cada asistente (48hs)
- [ ] Completé 8 almuerzos/cafés de negocios este mes

## FASE 6: Ganá el Día — Permanente

### Sistema de trabajo

- [ ] Imprimí y pegué mi Estándar Personal de Negocio
- [ ] Definí mi horario de jornada (inicio y fin)
- [ ] Establecí mi bloque de 2 horas diarias para generar leads
- [ ] Programé mis descansos

### Sistema de apoyo

- [ ] Encontré un Socio de Responsabilidad
- [ ] Acordamos día y hora fija semanal para reportar
- [ ] Identifiqué un Mentor y lo invité a almorzar
- [ ] Evalué si necesito un Coach profesional
